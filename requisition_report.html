The main object of the project was to load requisition and contract data from a SQL database (where data from another implemented product was loaded) into Power BI. In the previous product, it was inconvenient for the customer to work with primary documents, so the main requirement for the new product was the following: to allow opening files directly in Power BI in one click. The files were stored in the SQL database as a hex code.

First of all, it was necessary to encode the files in SQL and place them on the server in the required structure, so that it would be possible to implement the function of opening files in Power BI correctly. For encoding files I used Javascript and Node JS.

To implement the product, about 15 dimensions and 4 measures were used, as well as attributes with local links to files.
