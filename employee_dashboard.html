The main requirement of the customer was to build a dashboard with reports on employee activity for the sales department. The data source was SQL tables, Oracle database and API interfaces. Based on this data, a tabular model was built inside a PBIX file.

The dashboard consists of two pages: employee performance analysis and zone analytics (Employee level and Zone level).

All dashboard pages are adjusted by time-based filters and their main dashboards consist mainly of business cards and charts that reflect analytics on all KPIs.

On the "Employee Level" page, an employee rating system was developed where the user can customize coefficients and parameters to rate one or another employee.

A custom calendar visual element was also developed as part of this project. To build it, 37 elements were required, which contain business cards, shapes, and additional measures were built for the correct work of the calendar. Also, conditional formatting of each visual element was customized for the correct work of the calendar.

A tabular model consisting of about 8 dimensions and about 50 measures was used to construct the dashboard.
